IPA - Ionic Plugin Assistant

A NodeJs library that generates the necessary boilerplate for creating Ionic/Cordova plugins

IPA was made to make the average Ionic developers life a little bit easier, when wanting to create the own plugin for the Ionic Framework. Creating a plugin takes a LOT initial setup. I've written Medium articles about how to properly setup plugins, but I felt like even with that, the process was too convoluted for some.

So instead, I wanted to make the process easier - so I created the Ionic Plugin Assistant - IPA for short. Creating the package also made my life easier, because despite writing about the topic, I'm human and forget some of the steps, and often have to Google specifics - which lead me to my article.

When you run IPA, it will ask you a few questions about your plugin, such as: the plugin name, the authors name, github repo, license, etc... and then generate all of the boilerplate code, to the point that you could technically submit it to the Ionic Native platform as a new plugin - but it would probably be closed for lack of functionality.

IPA ensures that your plugin will adhear to Ionic plugin standards, common community standards, and set you up for success when it comes to making your plugin. This isn't to say that it will write your plugin for you, but it will give you a basic setup to get started with, where there was none to begin with.

The library generates 11 files, all with unique requirements to properly work in your specific platform & within the Ionic Framework

I opened-sourced IPA under the MIT license.

Languages

IPA is written in NodeJs and Javascript.

Services, Integrations and more

IPA utilizes Travis Continuous Integration for consistent deployment, Mocha for Unit Testing (with a 87% code coverage reported by CodeCov!), with Webpack and Babel used during the build phase.

NOLS - No One Likes Stylesheets

A NodeJs library that makes writing pixel perfect CSS a breeze

NOLS is a NodeJs package that I came up with after getting tired of making pixel-to-viewport conversions by hand, as in a large project you could potentially have dozens of small conversions that need to be done.

With NOLS at your side you can have the peace of mind of knowing that your pixel perfect CSS looks the same on all devices.

Other Node packages allow you to calculate a conversion, but none actually help you streamline your development process by applying the conversion in your stylesheets for you. Until NOLS.

What started out as a way to help myself out, has now become a pivotal part of my development process. As a result, I opened-sourced NOLS under the MIT license. So others could benefit as well.

Languages

NOLS is written in NodeJs and Javascript

Services, Integrations and more

NOLS utilizes Travis Continuous Integration for consistent deployment, Mocha for Unit Testing (with a 95% code coverage reported by CodeCov!), with Webpack and Babel used during the build phase.

Freedom Summer App - IOS & Android App

The Freedom Summer App (FSApp) is a location-based tour that uses a mobile device and GPS technology to transport you to 1964 where you train as a Freedom Summer volunteer on Miami University's campus in Oxford, Ohio.

Languages

FSApp is built using the open-source project ARIS, written in PHP and Objective-C. It leverages augmented reality to immerse the user into the rich history of the Freedom Summer, while using real-time image recognition to project historical images onto the user's screen at certain locations. The app is currently in Beta, and only available to pre-approved testers.
